The SNSS Field Employee Tracking App is a complete solution for managing field operations. Built with FlutterFlow and Firebase, it enables real-time employee location tracking, daily route planning, client and activity management, attendance monitoring, and performance reporting. The app supports multiple user roles—Admin, Manager, and Field Employee—with secure phone OTP authentication. Admins can create and assign employees, plan routes, and view live maps with geofencing. Field Employees can view assigned routes, update visit status, and submit reports. The backend uses Firestore, Cloud Storage, and security rules to ensure scalable, secure, and company-specific data handling, optimized for multi-organization use.
